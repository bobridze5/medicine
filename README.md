# Проект "Муниципальная больница"

Проект реализован в рамках курса от Yandex практикума (Python backend-разработчик) на фреймворке Django.

## Содержание сайта

1. Главная старница
2. Запись к врачу
3. Мои записи
4. Профиль (предоставляет просматривать данные пациента и изменять их)
5. Контакты
6. Новости
7. Регистрация / авторизация / смена пароля и прочие

## Функциональщина сайта
1. Многоступенчатая регистрация (Регистарация пользователя -> заполнение личных данных -> заполнение данных паациента)
2. Запись на приём
3. Отказ от приёма
4. Редактирование профиля (ФИО, номер телефона, почты, и тп)
5. Работа с новостями (способен создавать / удалять только администратор)
